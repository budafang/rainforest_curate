| 成果報告 項目 | 1.青年志工訓練過程與影響  行前完成本南文化資料搜集、本南話線上課程，先熟悉基本的問候用語。兒童營隊活動討論、行前規劃與討論會議，擬聚團隊共識。海外志工安全及性別平等課程，太陽能模組工作坊等。  2.計畫執行過程   1. 執行過程   **行前準備與採購 (8/4 - 8/9)**：  團隊先在市區採購糧食、水和物資，以避免消耗部落食物。搭乘吉普車前往部落，沿途經過 Penan 族人為抗議伐木而搭建的臨時路障。  **抵達與環境了解 (8/9)：**  車程約六時，傍晚抵達部落，當晚入住資源較匱乏的Long Tepen B區。  **核心服務執行 (8/10 - 8/14)：**  **太陽能系統安裝與交付：**  完成3塊太陽能板的搭建，為未來25戶準備了資源。挨家挨戶贈送並教導使用燈管、水壺、行動電源等設施。  **營隊活動調整：**  原定兒童營隊因小孩多住在A區學校且校方要求公文而取消。改辦理長青營隊，帶領老人家畫畫、書寫故事，並進行田野訪談，了解部落歷史。  **衛教與醫療協助：**  贈送不鏽鋼水壺並宣導塑化劑危害。贈送緊急醫藥箱給，製作英文/馬來文使用說明書。協助治療一位老人腳傷。教導老人家貝氏刷牙法，贈送牙刷和牙菌斑顯影劑。  **後續拜訪 (8/14 - 8/16)：**  拜訪位於美里的BMF，了解其協助Penan人搭建水管和接送學童的行動。  拜訪 Light Borneo 組織，了解其在亞庇地區協助原住民建置水力發電的經驗。   1. 資源整合及運用情形   這次計畫能順利進行，是多方資源協助的結果。我們每一次服務的村落是由馬來西亞當地的Keruan組織協助，包含安排交通、食宿等等。瑞士的BMF組織，則協助我們安裝太陽能板。我們的太陽能板是由馬來西亞的隆基 (Longi)公司贈送，但聽說因為川普關稅問題，目前經營的很困難，也許未來不見得會有太陽能板可以贈送了。最後，荒野基金會和我們中一翔鷹則負責實際執行的部份。另外，長安醫院附設居家護理所知道我們的行動，主動贊助我們此行四個團隊四個村落的所有牙刷。我們帶進去的木製玩具則是由玩具圖書館贊助的二手玩具。此行能夠順利成功，要感謝的人真的很多。 | | | |
| 成果報告 項目 | 過去，我們主要以義賣本南人手作工藝品的方式來募集資金，協助購買太陽能充電模組。今年，因為全部團隊成員都是高中三年級，課業壓力較大，我們沒有餘力進行義賣募款。感謝青年署核定的青年海外志工計畫，這個計畫的經費主要用在太陽能板的交通運輸（進去村莊的車資就花費5萬多元），聘請翻譯和當地購買灯具等等。  ![](data:image/png;base64...)   1. 執行成效   **太陽能系統建置**  本計畫最大的目標在協助本南人村落建置太陽能供電模組，在行前得知存在在美里的太陽能板被偷了一塊，因此，原訂四塊太陽能板模組只剩3塊太陽能板，大約可供1.13 kW的電源。  **營隊辦理**  原訂辦理兒童營隊，但因在村落的五天四夜，孩童全部留在學校，詢問校方，也無法前去分享交流。因此改為跟在地的長輩同樂。主要的活動有兩項，一則是邀請長輩畫出心中最珍貴的物件，有畫吹箭、刀具，也有畫地墊、巨嘴鳥、猴子等。聽長輩分享他們畫的物品，有的還有拿出實物供我們觀賞，相當溫馨。  另一個替代的活動是我們進行了大量的訪談，從生活的用品、民俗植物、料理方法到村落分裂及他們與伐木公司抗爭的歷史等等，透過這些訪談，我們未來若有機會再進雨林，能夠更貼切的修正我們的志工計畫。  **衛教宣導**  衛教宣導是本團隊今年新增的項目。由之前雨林計畫執行過程中，團隊發現本南人習慣用塑膠容器來裝熱飲，因此，今年我們進去雨林時特地準備了一些不銹鋼鍋具送給接待家庭，並向他們宣導熱水溶出塑化劑對人體的危害。  另外，本團隊有護理學生，由他來向村落的醫療照顧人員介紹如何使用醫藥箱。也現場幫有需要的村民進行簡單的傷口護理。這邊的物資相當缺乏，醫藥箱應該可以提供村民最基礎簡單的緊急處理。  最後，原訂要教導小孩貝氏刷牙法，因為孩童都不在，轉而教導長輩，由他們將牙刷轉贈給小孩。我們帶著長輩一起，透過實地示範，協助村民建立早晚刷牙的習慣，以減少蛀牙與牙周病的發生。  **回國分享**  本計畫原訂至少4次分享，但因計畫限制在回國2個月內核銷完畢預計會進行2次分享。其中一次已於8月30日在荒野親子團分享，共計約100人次。另一次，則於2025年10月11日台中圖書館烏日分館。  3.活動檢討、評估與反思 **詳細內容請參閱成果報告書**   1. 主要目標多已達成，唯一遺憾是兒童營隊未舉行。 2. 未來安裝太陽能板應預製鋁架，以克服木頭不平整的問題。 3. 部落以長者居多，兒童營隊雖取消，但長青營隊效果良好。 4. 進行衛教，發現醫療物資極缺，未來可更完善規劃並帶入更多物資。 5. 垃圾處理以掩埋為主，未來帶入物資應先規劃好垃圾處理方式。
