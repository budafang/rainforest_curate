# 為什麼要策展
如果你要講一件事，你可以寫書、拍電影。
策展則是像是導遊，安排路線，去最好玩的地方
安排路線 → 帶動情緒
安排景點 → 展現菁華
策展就是創造一種新的觀看方式

# Idea~~消失的森林
校園／圖書館快閃「被搬走的森林」
做法
一開始在公共空間擺滿紙做的葉片、動物剪影、綠色布條。每天固定時間撤掉一部分。經過的人會發現空間一天一天變空。
核心效果
不需要大展場，也能讓人感受到：環境消失常常不是突然，而是每天少一點。
可以加的互動
每被拿掉一塊，就留下標示：
這不是一棵樹而已
這也是棲地、路徑、記憶、水源

<!-- Slide number: 29 -->
# 空椅子計畫：替不能發聲的存在佔位
在展場或公共區域放一排空椅子，每張椅子代表一種存在：
一種消失中的物種
一個雨林村落的孩子
一條河流
一片森林
一種你從未認識的樹
椅子上放簡短故事卡。
核心效果
這種裝置很安靜，但很有力量。它會讓人感覺：有些位置原本是有人／有生命在的，但現在快要被看不見。
